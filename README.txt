项目名称：vivo商城(vue)
项目时间：30天左右
项目描述：
    技术栈：html，css，vue.js,vueX,session，vue-router,mint-ui,axios
    1.html，css，mint-ui框架完成静态页面的渲染，vue框架前后端的交互
    2.使用axios向后端发送请求获取数据，
    2.使用json模拟后端数据库传过来的数据
    2.完成的页面：首页，商品列表页，详情页，购物车页，地址页，订单页
    3.实现的功能：显示商品，添加到购物车，购物车商品结算，按条件显示商品等功能
项目遇到的问题：
    1.多个组件之间需要共用同一个数据，组件间通讯已不能满足需求
        解决：使用vuex公共数据库来管理数据
    2.不通过数据库来获取数据
        解决：使用json文件来模拟数据库数据的获取，session模拟数据的存储
项目收获：
    1.熟练vue及vuex的使用
    2.进一步json文件的格式
    3.对session的工作原理也有进一步的了解
